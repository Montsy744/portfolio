Mes outils préférés pour développer

Publié le 15 juillet 2025

Par Hugo Straseele

Quand on commence à coder, on se perd vite dans la tonne d’outils disponibles. Entre les éditeurs, les navigateurs, les extensions, les gestionnaires de versions… ça peut vite tourner à l’usine à gaz. Alors aujourd’hui, je te partage mes indispensables — simples, efficaces, et que j’utilise presque tous les jours.

VS Code

Mon éditeur de code préféré. Léger, puissant, plein d’extensions utiles (Prettier, Live Server, etc.), et super personnalisable. C’est un peu mon bureau de travail numérique.

Google Chrome + les DevTools

Rien de mieux pour tester et déboguer un site en temps réel. Les outils de développement intégrés sont juste indispensables pour comprendre ce qui se passe sous le capot.

Git + GitHub

Pour sauvegarder mon code, collaborer, ou revenir en arrière quand j’ai tout cassé (ça arrive). Même en solo, c’est un réflexe à prendre.

Mes extensions VS Code indispensables

Publié le 17 juillet 2025

Par Hugo Straseele

Visual Studio Code, c’est déjà super. Mais avec les bonnes extensions, ça devient vraiment un outil de travail ultra puissant. Voici celles que j’utilise tous les jours pour améliorer mon confort, mon code et ma productivité.

Prettier - Code formatter

Mon formatteur automatique préféré. Il s’occupe de l’indentation, des espaces, des virgules… Bref, il fait le ménage pendant que je code. Résultat : un code propre, cohérent et lisible sans y penser.

indent-rainbow

Chaque niveau d’indentation a une couleur différente. Simple, mais super pratique pour lire un code bien imbriqué (coucou HTML et les grosses fonctions JS).

Auto Rename Tag

Quand je modifie une balise HTML ou JSX, la balise fermante se met à jour toute seule. Un petit gain de temps, mais qui fait toute la différence quand on enchaîne les composants.

Live Server

Un clic, et mon projet s’ouvre dans le navigateur avec rechargement automatique à chaque sauvegarde. Parfait pour tester mes pages en temps réel sans prise de tête.

CSS Peek

Je peux survoler une classe CSS dans mon HTML pour voir directement son style défini ailleurs. Plus besoin de fouiller dans les fichiers, je gagne un temps fou.

vscode-pdf

Parfois, j’ai besoin de lire un PDF sans quitter l’éditeur. Cette extension me permet de le faire directement dans VS Code. Pratique pour garder tous mes outils au même endroit.

Git Graph

Un super visualiseur de branches Git. Idéal pour suivre l’historique de mon projet, voir les commits, les fusions, et mieux comprendre le flow Git.

Material Icon Theme

Des icônes stylées pour mes fichiers et dossiers. Ce n’est pas indispensable, mais ça rend l’interface plus agréable — et franchement, c’est toujours sympa de bosser dans un environnement joli.
