Mes Défis: Recréer des Sites

Je reprends des sites existants et je les réinvante pour m'entraîner.

Chaque projet est un défi technique et crêatif.

Refaire la page produit d'Amazon

Dans le cadre de mes exercices, j’ai recréé une page de type “Amazon” pour un casque de moto. L’objectif était de m’entraîner à concevoir une page d’annonce responsive et claire, en appliquant les bases du HTML, CSS et JavaScript.

J'ai travaillé sur :

  • La structure HTML pour organiser le contenu de manière logique.
  • Le CSS pour styliser la page et assurer une bonne expérience utilisateur.
  • Le JavaScript pour ajouter des interactions simples, comme un carrousel d'images.
  • La mise en page
  • L’affichage d’informations produit et photos
  • Une navigation simple et intuitive
